# PROJECT MAP

- `practica1/`: ejercicios de estructuras basicas (pilas, colas, radio).
- `practica2/`: ejercicios sobre pilas, colas, lectura de ficheros y recorridos/analisis de playlists y radios.
- `practica3/`: ejercicios posteriores con listas y colas; ahora contiene tambien las copias locales necesarias para compilar y empaquetar la entrega de forma autocontenida.
- `other_files/`: material auxiliar, incluido el enunciado de la practica 2.
- `music.c`, `music.h`, `radio.c`, `radio.h`, `types.h` en la raiz: implementaciones compartidas que reutiliza `practica2/`.
